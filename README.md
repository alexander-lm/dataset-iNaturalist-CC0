# Dataset: PestNeuroVision/iNaturalist

This repository contains a compiled collection of insect images for pest detection. The dataset is published as supplementary material for the paper: **"PestNeuroVision: A mobile application based on convolutional neural networks and computer vision for agricultural pest detection in the Cañete Valley, Lima, Peru."**.

## Data Sources and Licensing

The images in this repository were sourced from **iNaturalist** under the **CC0 1.0 Universal (Public Domain)** license. This license permits redistribution, modification, and use of the data without legal restrictions.

To ensure transparency and facilitate the validation of the original sources, the compiled dataset is based on the following collections:

* ***Ceratitis capitata*** **(adult):** [Link](https://www.inaturalist.org/taxa/199404-Ceratitis-capitata/browse_photos?photo_license=cc0)
* ***Dione juno*** **(adult)**: [Link](https://www.inaturalist.org/taxa/130451-Dione-juno/browse_photos?photo_license=cc0&term_id=1&term_value_id=2)
* ***Dione juno*** **(larva)**: [Link](https://www.inaturalist.org/taxa/130451-Dione-juno/browse_photos?photo_license=cc0&term_id=1&term_value_id=6)
* ***Ligyrus gibbosus*** **(adult)**: [Link](https://www.inaturalist.org/taxa/1430438-Ligyrus-gibbosus/browse_photos?photo_license=cc0)
* ***Spodoptera frugiperda*** **(adult)**: [Link](https://www.inaturalist.org/taxa/132468-Spodoptera-frugiperda/browse_photos?photo_license=cc0&term_id=1&term_value_id=2)
* ***Spodoptera frugiperda*** **(larva)**: [Link](https://www.inaturalist.org/taxa/132468-Spodoptera-frugiperda/browse_photos?photo_license=cc0&term_id=1&term_value_id=6)

## Usage

This compilation is intended for research and academic purposes. If you use this dataset in your work, please cite the original paper and the sources mentioned above.
